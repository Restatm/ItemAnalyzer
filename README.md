# Item Analyzer Dashboard

ESG 문항 개정 및 문항 시뮬레이션을 위한 데이터 사이언스 기반 대시보드입니다.

## 주요 기능
- **문항 분석**: 유형별 득점률 및 양극화 분석
- **시뮬레이션**: 문항 배점 조정 시의 유형별 CV 및 기업별 득점 변화 예측
- **데이터 시각화**: 3D 공간 좌표 기반의 문항 분포 시각화

## 사용 방법
1. 웹 사이트에 접속합니다.
2. 표준화된 평가 결과 엑셀 파일(`.xlsx`)을 업로드합니다.
3. 실시간으로 분석 결과와 시뮬레이션 결과를 확인합니다.
